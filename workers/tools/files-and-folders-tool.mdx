## Overview

The files and folder tool enables your workers to create, organize, and manage files directly within Utari. This powerful capability allows your workers to generate content, save it in organized folder structures, and provide you with easy access to download, copy, or export your files.

## Enabling the Files and Folder Tool

<Steps>
  <Step title="Navigate to Worker Tools">
    Select your worker and click on the **Tools** tab.
  </Step>
  <Step title="Locate Files and Folder Tool">
    Scroll through the available tools to find **Files and Folder**.
  </Step>
  <Step title="Enable the Tool">
    Check the box next to Files and Folder to activate it.
  </Step>
  <Step title="Review Capabilities">
    By default, all six capabilities (create, delete, edit, read, rewrite, replace) are enabled. You can customize which capabilities to enable based on your needs.
  </Step>
  <Step title="Save Configuration">
    Your changes are automatically saved. The worker can now create and manage files.
  </Step>
</Steps>

<Note>
  The files and folder tool is essential for workers that need to generate deliverable content like documents, reports, summaries, or any structured output.
</Note>

## Creating Files with Folders

### Basic File Creation

<Steps>
  <Step title="Start a Chat">
    Open a conversation with a worker that has the files and folder tool enabled.
  </Step>
  <Step title="Request File Creation">
    Ask your worker to create a file with specific content. Include instructions to organize it in a folder:

    ```
        Please create a file on mastering VSLs. Include a summary and place the file into a folder called "VSL Mastery".
    ```
  </Step>
  <Step title="Watch File Generation">
    The AI worker will:

    - Interpret your instructions
    - Generate the requested content
    - Create the folder structure
    - Save the file in the specified location
  </Step>
  <Step title="Access Your File">
    Navigate to the **Tasks** folder in your workspace to locate the newly created content.
  </Step>
</Steps>

### Example Request Patterns

<CodeGroup>

```text Organized Content Creation
Create a marketing plan document and save it in a folder called "Q4 Campaign Planning".
```


```text Multiple Files
Generate three blog post drafts on AI trends and organize them in a "Blog Content" folder.
```

```text Structured Documentation

```



## Accessing and Managing Files

Once files are created, you have multiple options for working with them:

<Steps>
  <Step title="Navigate to Tasks Folder">
    Click on the **Tasks** folder in your Utari workspace to view all files created by workers.
  </Step>
  <Step title="Open Your Folder">
    Find and open the specific folder containing your file (e.g., "VSL Mastery").
  </Step>
  <Step title="Access File Options">
    Click on your file to reveal available actions:

    - **Copy**: Copy content to clipboard
    - **Download**: Save file to your computer
    - **Export as PDF**: Generate a PDF version
    - **Edit**: Modify the file content
    - **Delete**: Remove the file
  </Step>
</Steps>

<Info>
  Files created by your workers are stored in your Utari workspace and remain accessible across sessions until you delete them.
</Info>

## File Management Operations

### Reading Files

Ask your worker to read and reference existing files:

```text
Read the VSL Mastery file in the VSL Mastery folder and summarize the key points.
```

### Editing Files

Request modifications to existing files:

```text
Edit the marketing plan in the Q4 Campaign folder and add a section on social media strategy.
```

### Rewriting Files

Completely regenerate file content:

```text
Rewrite the blog post draft with a more casual, conversational tone.
```

### Replacing Files

Substitute files with new versions:

```text
Replace the old proposal template with a new version that includes our updated pricing.
```

### Deleting Files

Remove unwanted files or folders:

```text
Delete the draft files in the Archive folder.
```

## Export and Download Options

### Copy to Clipboard

Perfect for quick content transfer:

- Click the file
- Select **Copy**
- Paste into any application

### Download File

Save the original file format:

- Click the file
- Select **Download**
- File saves to your downloads folder

### Export as PDF

Create professional, shareable documents:

- Click the file
- Select **Export as PDF**
- PDF generates with formatting preserved

<Tip>

## Troubleshooting

<AccordionGroup>
  <Accordion title="File not appearing in folder">
    Verify that:

    - The files and folder tool is enabled
    - You specified a folder name in your request
    - You're looking in the correct folder location
    - The worker completed the file creation successfully
  </Accordion>
  <Accordion title="Can't find the Tasks folder">
    Check:

    - You're logged into the correct Utari account
    - Your workspace view is showing the files section
    - Try refreshing your browser
    - Navigate to your dashboard and look for the workspace files area
  </Accordion>
  <Accordion title="Unable to download or export files">
    Try:

    - Checking browser permissions for downloads
    - Disabling popup blockers temporarily
    - Using a different browser
    - Ensuring the file completed generation before attempting download
  </Accordion>
  <Accordion title="File content is incomplete or incorrect">
    Consider:

    - Providing more detailed instructions in your prompt
    - Referencing knowledge base documents for guidance
    - Asking the worker to rewrite the file with clearer requirements
    - Breaking complex content into multiple smaller files
  </Accordion>
  <Accordion title="Folder organization is messy">
    Best practices:

    - Always specify folder paths in your requests
    - Create a folder naming convention and stick to it
    - Regularly clean up and reorganize outdated content
    - Use nested folders for complex project structures
  </Accordion>
</AccordionGroup>

## Tips for Effective File Management

### Clear File Requests

❌ **Vague Request:**

```text
Create a document about marketing.
```

✅ **Specific Request:**

```text
Create a comprehensive marketing strategy document covering Q4 goals, target audiences, and channel recommendations. Save it in "Marketing/Strategy/Q4-2024".
```

### Maintaining Organization

❌ **Poor Organization:**

```text
Create some blog posts.
```

✅ **Well-Organized:**

```text
Create three blog posts about AI automation (beginner, intermediate, advanced) and save them in "Blog Content/Series/AI Automation/Drafts".
```

### Version Management

❌ **No Version Control:**

```text
Rewrite the proposal.
```

✅ **With Version Control:**

```text
Create an updated version of the proposal in "Clients/ABC/Proposals" and name it "Proposal_v3_Final.md".
```
